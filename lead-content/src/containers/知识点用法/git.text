
	1.命令提示符（后台管理员中使用）
nvm 的下载地址 https://github.com/coreybutler/nvm-windows/releases   点击nvm-setup.exe    
	下载新的node。js版本
	nvm install 版本号
	nvm ls available  查看有哪些版本
	nvm use 想要应用的版本号
	nvm ls  查看当前所用的版本号
	nvm uninstall 卸载的版本号
